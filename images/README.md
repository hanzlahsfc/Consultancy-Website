# Images folder

Drop your images in here with these filenames so the site picks them up automatically:

## Main folder
- `logo.png` — your company logo (transparent PNG, ~200x80px)
- `hero-bg.jpg` — homepage hero background (aircraft photo, 1920x1080)
- `about-hero.jpg` — About page hero banner
- `services-hero.jpg` — Services page hero banner
- `fleet-hero.jpg` — Fleet page hero banner
- `contact-hero.jpg` — Contact page hero banner

## `brands/` subfolder (optional — currently shows text logos)
- `piper.png`, `beechcraft.png`, `cessna.png`, `flight-design.png`, `socata.png`, `elixir.png`, `mooney.png`, `rotax.png`, `lycoming.png`

## `services/` subfolder (optional)
- Images referenced in `services.html` — currently uses placeholder cards

## `fleet/` subfolder (optional)
- Aircraft photos referenced in `fleet.html` — currently uses placeholder cards

Until you drop images in, all sections show clean placeholder boxes with "Replace with…" hints so the layout still works.
